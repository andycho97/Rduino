Data Analysis

We write scripts in a variety of languages (R, Python, Processing) to handle interesting analysis problems from data read in from our Arduinos.

Hardware

We use the trendy open-source Arduino boards for collecting environment data and presenting curated audio and visual output.

Full Integration

We are working to create an R package called Rduino that fully integrates the R programming language with the Arduino board.
